# Задача 18: Интеграция навигационного колеса (OptionWheel) — ВАРИАНТ Б (ВЫБРАН)

Требуется интегрировать компонент `OptionWheel` в качестве центрального интерактивного контроллера для переключения услуг или стека.

**Концепция (Вариант Б - Центральный контроллер):**
Колесо располагается по центру секции `Services` или `About`. Пользователь вращает его (драг / колесико мыши), переключая категории. Выбранный элемент динамически меняет соседний парящий блок текста.
- **Стиль**: Крупный геометрийский центр, тончайшие кольца (1px outline, 15% opacity), минималистичный стиль в Void-эстетике.

**Шаги выполнения:**
1. Открой компонент `src/components/OptionWheel.jsx`.
2. Очисти стили от градиентных и серых плашек, переведи на 1px контурные линии на чистом черном фоне (`#000000`).
3. Встрой колесо в секцию `Services` (или `About`) в `src/app/page.tsx`.
4. Настрой передачу выбранного индекса в локальное состояние или сервис для обновления контекстного текста.
5. Проверь поддержку сенсорных устройств и колесика мыши.
6. Сделай коммит: `git commit -am "feat:OptionWheel central interactive controller integration"`.
