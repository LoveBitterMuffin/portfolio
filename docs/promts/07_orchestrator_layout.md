# Промт 7: Интеграция оркестратора и наполнение контентом секций

Объединить сервисы `ScrollInterfaceService`, `BackgroundGraphicsService` и `VideoInteractionService` в центральном оркестраторе `src/app/page.tsx`, загрузить контент разделов и настроить верстку 5 панелей.

## Инструкции по реализации

### 1. Подготовка контента
*   Преобразуйте контент из [CONTENT.md](file:///c:/mp/portfolio/docs/CONTENT.md) в структурированный JSON-файл `src/data/content.json` или в константный объект внутри проекта для легкого импорта в React-компоненты.
*   Контент должен содержать данные для всех 5 секций: `Intro`, `Education`, `Experience`, `About & Stack`, `Contacts`.

### 2. Реализация оркестратора (`src/app/page.tsx`)
В клиентском компоненте главной страницы:
*   Инициализируйте ссылки `useRef` для DOM-элементов: `scrollWrapperRef`, `horizontalTrackRef`, `canvasRef`, `videoRef`, `videoContainerRef`.
*   В хуке `useEffect` выполните инициализацию трех сервисов:
    1.  `BackgroundGraphicsService` (связывается с контейнером и канвасом).
    2.  `VideoInteractionService` (связывается с видео и его контейнером).
    3.  `ScrollInterfaceService` (принимает скролл-враппер, трек, массив панелей и коллбэк изменения секций).
*   В коллбэке `onSectionChange(activeSection, sectionProgress)`:
    *   Обновляйте активный шаг в UI (например, состояние `activeStep` для отображения в углу `[STEP.0X/05]`).
    *   Вызывайте `BackgroundGraphicsService.morphTo(activeSection, sectionProgress)` для плавного морфинга частиц при прокрутке.
    *   Передавайте прогресс скролла в `VideoInteractionService.updateScrollProgress(globalProgress)` или обновляйте состояние видео.
*   Добавьте глобальный слушатель `pointermove` (или `mousemove` + `touchmove`), который:
    *   Передает координаты мыши на Canvas: `bgService.updatePointer(e.clientX, e.clientY)`.
    *   Передает координаты мыши в видео-сервис: `videoService.updatePointer(e.clientX, e.clientY)`.
*   **ResizeObserver**: Создайте ResizeObserver для отслеживания размеров окна и вызова методов `.resize(w, h)` у сервисов графики.
*   **Очистка (Cleanup)**: Возвращайте функцию очистки в `useEffect`, которая вызывает `.destroy()` у всех трех инстансов сервисов и удаляет глобальные слушатели событий.

### 3. Верстка и контент панелей
Отрендерите 5 панелей внутри горизонтального трека:
*   **Intro**: Имя большими буквами Space Grotesk, роль, подпись и кнопка CTA. Сетка частиц здесь должна мягко обтекать силуэт.
*   **Education**: Рендеринг 3 блоков обучения. Частицы Three.js на фоне перестраиваются в Detroit концентрические кольца.
*   **Experience**: Таймлайн горизонтального типа с 3 блоками опыта работы.
*   **About / Stack**: Grid с перечислением навыков (Frontend Core, Backend & DevSecOps, Soft Skills) и биографический текст. Частицы собираются в 3D-сферу.
*   **Contacts**: Форма обратной связи, почта, социальные ссылки. Частицы рассеиваются.
*   *Эффект параллакса контента*: Для текстовых блоков задайте атрибут `data-speed` (например, от `0.1` до `0.5`). В `ScrollInterfaceService` анимируйте горизонтальное смещение этих блоков с разной скоростью относительно движения панели для создания эффекта глубины.

## Критерии приемки
1. Проект компилируется и билдится без ошибок сборщиком.
2. При скролле все три сервиса работают синхронно: трек движется горизонтально, частицы морфятся в соответствующие 3D-фигуры, видео силуэта человека реагирует на скролл/мышь.
3. Полностью отсутствует реактивный стейт для передачи мыши и прогресса кадров, FPS стабильно держится на 60 кадрах/с на средних компьютерах.
4. Все ресурсы освобождаются при уничтожении компонентов.
