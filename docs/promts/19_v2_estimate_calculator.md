# Задача 18: Интерактивный калькулятор сметы (Services)

Требуется создать виджет для динамического расчета стоимости и сроков проекта.

**Шаги выполнения:**
1. Создай компонент `src/components/EstimateCalculator.tsx`.
2. Реализуй интерактивные toggle/select опции: Тип сайта (Landing/Web App), UI/UX (Шаблон/Кастом), ИИ/Чат-бот (Да/Нет), DevSecOps (База/Pro).
3. Добавь GSAP-анимацию счетчика для плавного пересчета цены (CountUp effect).
4. Замени текущий статический список цен в `page.tsx` на этот компонент.
5. Настрой кнопку `Отправить смету`, которая будет формировать строку для `mailto:` или Telegram с выбранной конфигурацией.
6. Сделай коммит: `git commit -am "feat: interactive estimate calculator widget"`.
