# Задача 21: Cookie & 152-ФЗ Compliance Banner

Внедрение футуристичного Glassmorphism баннера для согласия на обработку персональных данных (ФЗ РФ № 152-ФЗ).

**Шаги выполнения:**
1. Создай компонент `src/components/CookieConsent.tsx`.
2. Оформи его в виде плавающей панели снизу с эффектом backdrop-blur (стекло).
3. Текст должен ссылаться на 152-ФЗ и уведомлять об использовании cookie/аналитики.
4. Добавь кнопки "Принять" и "Политика конфиденциальности".
5. Сохраняй выбор пользователя в `localStorage` (ключ `cookie_consent=true`), чтобы не показывать баннер повторно.
6. Вставь компонент в `src/app/layout.tsx`.
7. Сделай коммит: `git commit -am "feat: cookie 152-fz compliance banner"`.
