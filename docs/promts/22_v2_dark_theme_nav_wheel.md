# Задача 22: Темная тема (Dark Mode) и Навигационное Колесо

Требуется адаптировать весь сайт под темную тему в стиле Dala (Void) и лаконично вписать навигационное колесо (`OptionWheel`).

**Интеграция навигационного колеса (Дизайн-концепция):**
Чтобы колесо не ломало стиль «парящих элементов в пустоте» (Void Layout), оно не должно иметь сплошных фонов, карточек или толстых рамок.
- **Стиль**: Тончайшие концентрические круги (1px outline, opacity 10-15%). 
- **Типографика**: Ультратонкий шрифт для неактивных пунктов (opacity 40%), активный пункт подсвечивается чисто белым (или акцентным фиолетовым/синим).
- **Расположение**: Зафиксировать его в нижнем левом углу как "компас" навигации по сайту, либо использовать как центральный контроллер в секции Services вместо стандартного скролла.

**Шаги выполнения:**
1. Обнови CSS переменные в `src/app/globals.css`.
   - Основной фон: Чистый черный (`#000000`).
   - Текст: `Bone White (#ffffff)`.
   - Неактивный текст: `Ash Gray (#9a9a9a)`.
2. Интегрируй (или отрефакторь) компонент `OptionWheel`.
   - Убери любые `bg-gray-100` или тени. 
   - Сделай так, чтобы он сливался с черным фоном, оставляя лишь геометрический скелет.
3. Проверь контрастность контурных треугольников (Three.js частицы) на черном фоне. Они должны стать ярче (Electric Iris / Accent Blue).
4. Настрой плавный GSAP fade-in для колеса при загрузке.
5. Сделай коммит: `git commit -am "feat: dark theme integration and minimalist nav wheel"`.
